## Required entry template

Every entry contains: purpose, relevant files, entry points, call graph, dependencies, related modules, important structs, and important traits. Add ownership, concurrency, and failure notes where applicable.

All source paths and symbols must link to the official [Anza Agave repository](https://github.com/anza-xyz/agave) under the rules in [SOURCE-POLICY.md](../SOURCE-POLICY.md). Index entries record a verification date or commit and flag stale line anchors.
